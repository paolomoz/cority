Partnerschaft mit IH Data Analyst verbessert Expositionsbewertungen

Cority Software

style
navy
  • Zuletzt aktualisiert: 29. Juni 2026

Arbeitshygieniker können dank einer neuen Schnittstelle zwischen Medgate und IH Data Analyst nun besser fundierte Entscheidungen über das Risiko von Expositionen gegenüber Chemikalien, Lärm und anderen Gefahren am Arbeitsplatz treffen.

Medgate-Kunden können dank einer im vergangenen Jahr angekündigten exklusiven Partnerschaft nun direkt auf IH Data Analyst zugreifen, der von EASi entwickelt wurde. Er ermöglicht es Nutzern von Medgates Software für industrielle Hygiene, im Rahmen ihrer Risikobewertungen eine Bayessche Entscheidungsanalyse (Bayesian Decision Analysis, BDA) durchzuführen, um klarere Ergebnisse zu Expositionen bei begrenzten Daten zu erhalten.

IH Data Analyst ist weithin als Branchenstandard bei BDA-Tools anerkannt und wird von Hygienikern auf der ganzen Welt verwendet. Bislang mussten Fachleute für industrielle Hygiene eine eigenständige Anwendung nutzen, was ihre Risikoanalyseprozesse erheblich zeit- und arbeitsaufwendiger machte. Durch die Integration dieses Tools in Medgates Suite für industrielle Hygiene werden Arbeitshygieniker in der Lage sein, alle ihre Risikobewertungen an einem einzigen, zentralen Ort zu verwalten.

Medgates Product Managerin für industrielle Hygiene, Monica Melkonian CIH, sagte: „Die Durchführung der Bayesschen Analyse mit IH Data Analyst wird es Medgate-Kunden ermöglichen, ihr fachliches Urteil in Bezug auf das Risiko zu validieren, um sicherzustellen, dass die bestmöglichen Entscheidungen über das Risiko von Mitarbeiterexpositionen getroffen werden.“

Über EASi

EASi entwickelt IH Data Analyst (IHDA), ein innovatives, einzigartiges Produkt zur Analyse arbeitsbezogener Daten, das den Arbeitshygieniker bei der Interpretation von Datensätzen mit geringer Stichprobengröße unterstützt.
